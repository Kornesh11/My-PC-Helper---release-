# My-PC-Helper---release-
Моя первая программа на языке Python, которая немного облегчает жизнь обычному пользователю компьютера
На экранке сразу видно 3 кнопки: Zapret Оптимизация и TG-PROXY
что бы работали 1 и 3 кнопки надо в настройках (в правом верхнем углу) указать путь до файла .bat
дальше идут Insert - скрывать все рабочие ярлыки на рабочем столе
End - скрывать все окна кроме самого окна прогрммы и Chrome
home - открывает вкладку гугла
